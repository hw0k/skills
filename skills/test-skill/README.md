# Test Skill

A simple test skill demonstrating the basic skill structure.

## Installation

```bash
# From local directory
npx skills add ./skills/test-skill

# From GitHub (future)
npx skills add hw0k/skills --skill test-skill
```

## Usage

Invoke explicitly:
```
/test-skill
```

Or ask Claude:
```
"Can you review this code using the test skill?"
```

## What This Demonstrates

- SKILL.md structure with frontmatter
- Principle organization
- Example code blocks
- Checklist format
- When the skill is applied
